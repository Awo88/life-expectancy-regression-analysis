Life Expectancy
Regression Analysis

Statistical modeling of global life expectancy trends using WHO data, with regression analysis and predictive analytics across countries and time.

Overview

This project uses regression analysis to model and predict life expectancy across countries using World Health Organization data. By examining the relationship between life expectancy and key socioeconomic and health indicators — GDP, healthcare access, education, disease prevalence — the model identifies which factors most strongly predict population health outcomes.

Built in R, the analysis applies multiple regression techniques to a global dataset, producing a predictive model with interpretable coefficients and out-of-sample validation metrics.

Methods
Multiple Regression
OLS and regularized regression to model life expectancy as a function of health and economic predictors
WHO Data Analysis
Real global health data across 190+ countries cleaned, transformed, and used for model training
Predictive Analytics
Out-of-sample forecasting with RMSE and R² evaluation across model specifications
Statistical Modeling
Residual analysis, multicollinearity checks, and feature selection for model refinement
